### First Week's Problem set

Initialize variables and cout them.

```cpp
#include <iostream>

using namespace std;

int main () {
  int age = 20;
  float marks = 979;
  char grade = 'A';
  string name = "Abdul Samad";

  cout
    <<"My name is "
    <<name
    <<'.'
    <<" I'm "
    <<age
    <<" years old"
    <<'.'
    <<" I got "
    <<marks
    <<" marks out of 1100."
    <<" My Grade is "
    <<grade
    <<'.'
    <<'\n';
}
```
